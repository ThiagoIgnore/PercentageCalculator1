Percentage Calculator


How to use

This is a percentage calculator with two operation buttons.
Each button launch one javascript function with one especific operation.
Examples:
If you want to know how much percent 200 is from 2340 you should set 200 as the x value and 2340 as y value and press the "X Is What Percent Of Y" button.
If you want to know how much is 0.82 percent of 4000 you should set 0.82 as x value and 4000 as y value and press the "X Percent Of Y Is" button.
